# デスクトップアプリ操作確認フロー

このメモは、採用担当者・レビュー担当者がスクリーンショットだけでなく「どの操作ができるか」を短時間で確認できるようにした動作確認手順です。

## 定型文管理アプリ

想定用途: 問い合わせ返信、初回受付文、案内文、確認項目のテンプレート管理。

1. `python teikei_kanri.py` で起動する。
2. 新規テンプレートとして `PC不調の初回確認` を作成する。
3. 本文に `発生時刻、対象アプリ、再現条件、エラー表示を確認します。` を入力して保存する。
4. 一覧からテンプレートを選択し、コピー操作を行う。
5. エディタやメール本文へ貼り付け、文面がそのまま転記されることを確認する。

確認できること:

- 定型文の作成・編集・削除というCRUD操作
- JSON / テキストファイルによるローカル保存
- ワンクリックコピーによる問い合わせ対応時間の短縮

## 付箋アプリ

想定用途: 対応中チケット、確認待ち、折り返し連絡、作業メモの一時管理。

1. `python sticky_notes.py` で起動する。
2. `共有フォルダ権限確認` という付箋を作成する。
3. 付箋を画面端へ移動し、本文に確認項目を入力する。
4. アプリを終了して再起動する。
5. 付箋の位置と本文が復元されることを確認する。

確認できること:

- 複数付箋の作成・移動・削除
- JSON保存による状態復元
- 対応中メモを残すときの注意点を説明できること

## 注意

本番業務で使う場合は、パスワード、個人情報、顧客情報、社内固有情報を付箋やテンプレートに残さない運用ルールが必要です。
